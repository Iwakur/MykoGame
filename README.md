# МикоГео

МикоГео — невелика вебгра українською мовою для вивчення урбанонімів Миколаєва. Ідея схожа на Seterra Geography: гравець дивиться на мапу міста, вчить назви місцевостей, а потім проходить тест кліками по реальній OpenStreetMap-мапі.

## Режими

- **Навчання** — усі локації показані маркерами; клік відкриває назву, район і коротку підказку.
- **Тест** — гра випадково просить знайти урбанонім; після кліку зберігає результат, показує похибку в метрах і в кінці лишає список правильних та неправильних відповідей.

## Урбаноніми в першій версії

Соляні, 100-й квартал, Військова Слобідка, Слобідка, Абіссинія, Ліски, Намив, Сухий Фонтан, Старий Водопій, Новий Водопій, ПТЗ, Ракетне Урочище, Тернівка, Темвод, Бормашина.

Координати в `src/locations.js` підібрані як орієнтовні центри місцевостей у межах Миколаєва. Для історичних або розмовних назв без чітких адміністративних меж використано наближені точки, щоб гра була одразу придатна для навчання.

## Запуск

```bash
npm run start
```

Після запуску відкрий http://localhost:5173/. Проєкт не потребує збирання: це статичні HTML, CSS і JavaScript-файли.

## Перевірка

```bash
npm test
```

Тест перевіряє кількість локацій, унікальність і те, що координати лежать у межах Миколаєва.
